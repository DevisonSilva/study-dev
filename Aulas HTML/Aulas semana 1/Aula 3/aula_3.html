Material Apoio

Criação de Link: Quando queremos criar um link, sempre utilizaremos a tag <a href=""> onde HREF sempre será um atributo obrigatório, que indicará o link para onde vai e o children será o texto que aparecerá para o usuário clicar.

Criação de Imagem: Quando quisermos criar uma imagem, utilizaremos a seguinte tag <img src="" alt=""> onde SRC (source) será o caminho da imagem e ALT (alternative) será o texto alternativo da imagem, que será visível se a imagem não carregar ou se o usuário estiver utilizando um leitor de tela (deficiência visual).
